# _quarantine/

Files moved here are **not deleted** — they're parked because they aren't part of the active production codebase. If you're missing a file, check here first.

Quarantined on: 2026-04-15

## What's in here

### `relfar-reverse-engineering/` (20 scripts)
Probe, scan, and sniff scripts written during the Relfar V4 protocol reverse-engineering phase (April 2026). Includes the legacy RS485/Modbus V3 server (`_legacy_rs485_server.py`), BLE scanner, Wi-Fi connector, port scanners, register probes, and packet sniffers. None are imported by the production `relfar_server.py`. Valuable if you ever need to re-probe the controller or debug the protocol.

### `relfar-scan-artifacts/` (6 files)
Output files generated by the probe scripts above: JSON result dumps, a register scan log, and a Wireshark pcap capture of the controller boot sequence.

### `branding-assets/` (6 files)
Prisma logo variants (PNG + SVG, light + icon). Not referenced in any service code. Kept in case they're needed for marketing or docs.
